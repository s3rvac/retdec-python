Welcome To retdec-python's Documentation!
=========================================

`retdec-python <https://github.com/s3rvac/retdec-python>`_ is a Python library and tools providing easy access to the `retdec.com <https://retdec.com>`_ decompilation service through their public `REST API <https://retdec.com/api/>`_.
